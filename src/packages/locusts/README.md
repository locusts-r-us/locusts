## Description

Locusts is a series of libraries that allows you to introduce locusts into your
codebase. Why would you want to do that? I don't know.

## Installation

```sh
npm add locusts
```

## Example

```typescript
import introduceLocusts from "locusts";

function main() {
  introduceLocusts();
  // Locusts are now in your codebase!
}
```
