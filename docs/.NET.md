# Locusts

## Description

Locusts is a library that provides a way to introduce locusts into your code.

Why would you want to do that? I don't know.

## Installation

```sh
dotnet add package Locusts
```

## Example

```cs
using Locusts;

Locusts.Introduce();
// Locusts are now in your codebase!
```
